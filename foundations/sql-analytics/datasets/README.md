# Datasets

Ce dossier contient les fichiers SQL utilisés pour créer et alimenter la base de données de démonstration de la section `sql-analytics`.

## Contenu

- `schema.sql` : script de création des tables
- `seed.sql` : script d’insertion des données d’exemple

## Vue d’ensemble du dataset

Le dataset simule un environnement e-commerce simplifié avec les entités suivantes :

- clients
- catégories
- produits
- commandes
- lignes de commande
- paiements

## Objectif

Ce dataset sert de base pour pratiquer et démontrer des compétences SQL fondamentales en analyse de données, notamment :

- le filtrage
- les jointures
- les agrégations
- le calcul de KPI
- l’analyse orientée métier

## Remarque

Les données sont fictives et ont été créées dans le cadre de ce portfolio.s

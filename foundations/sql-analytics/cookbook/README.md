# SQL Cookbook

Ce dossier regroupe des exemples pratiques de requêtes SQL, organisés par thème.

L’objectif est de démontrer les compétences SQL essentielles pour l’analyse de données, avec un accent sur la lisibilité, la logique métier et les patterns analytiques les plus courants.

## Contenu

- `01_basic_select_filter.sql` : exemples de requêtes de base avec sélection, filtrage, tri et limitation
- `02_joins.sql` : exemples de jointures entre plusieurs tables

## Objectifs

Ce cookbook montre comment utiliser SQL pour :

- extraire des données
- relier plusieurs tables
- préparer des jeux de données exploitables
- répondre à des questions métier simples

## Remarque

Chaque fichier contient des requêtes commentées avec un objectif d’apprentissage précis.

Des exemples plus avancés seront ajoutés progressivement, notamment sur :

- les agrégations
- `CASE WHEN`
- les CTE
- les fonctions de fenêtre
- l’analyse temporelle
- le calcul de KPI
- les contrôles qualité des données
